Hierarchy of a web page

Page Elements

Page

  • The top level entitiy in a web page

Layout

  • Sub sections of the page E.G. Header, Sidebar, Main.

Module (optional)

  • A collection of components that solve a specific task, good for large applications and you need to have a way to group components that solve the same thing.

Component

  • Self contained in nature, they adhere to the single responsibility principal, A component does one thing, it solves a specific task.
